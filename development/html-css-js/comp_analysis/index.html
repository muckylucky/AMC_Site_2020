“Liberate Your Data”

Our customer data platform liberates your first-party data so you can orchestrate individual experiences in every stage of the customer lifecycle.

Visit the site

TL:DR

Primarily a CDP (Customer Data Platform): gathering data from a variety of sources to enable powerful segmentation.

They do not provide an end-to-end solution in the way that SMC does e.g no native content creation solution.

They appear to be all about the data and do well with the real-time piece but lack the batch audience segmentation necessary for omni-channel marketing.

Posessing a strong AI/Machine Learning powered recommendations engine for use with the clients web site and Customer Lifecycle Orchestrations which SMC largely lacks but feature-wise SMC provides the average marketer much more possibilities within an integrated solution.

Blueconic’s focus is on real-time consumer data consolidation (they call universal profile) whereas we are focused on multi-channel execution. However we were hearing a lot about them from CSMs.

TL:DR

Site is an extremely powerful addition to the SMC suite of products with it’s core capabilites being to provide real-time personalisation, creation and tracking of anonymous profiles and analytics providing insights into visitor behaviour that can be leveraged for the creation of 1 to 1 tailored online experiences.

However it does face numerous challenges in the current market:

  • Many clients have the Site product but are not utilising it because it is very technical and has a steep learning curve. Typically those clients that do use Site a lot are doing so via partners.
  • The Site UI is clunky and unintuitive and the information architecture is somewhat confusing.
  • A lack of major updates in the last few years means that Site is lacking some features that are provided by other competitors.
  • A separate database from the rest of SMC causes technical problems and means that Site data is only made available in SMC on a daily basis. This also poses challenges for the proposed integration of Site into SMC.
